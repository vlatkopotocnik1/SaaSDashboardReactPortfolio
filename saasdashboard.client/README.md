# SaaS Dashboard (React)

## Brzi start

Preduvjet: Node.js 20+.

```bash
npm install
npm run dev
```

## Skripte

- `npm run build` - produkcijski build
- `npm run lint` - ESLint provjera (bez warninga)
- `npm run lint:fix` - ESLint autofix
- `npm run format` - Prettier formatiranje
- `npm run format:check` - Prettier provjera
